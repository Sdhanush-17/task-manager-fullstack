# Task Manager (Full Stack)

A full-stack task management application that allows users to register, login, and manage tasks securely.

1. Tech Stack
- React
- Node.js
- Express
- MongoDB
- JWT Authentication

2. Features
✔ User Authentication  
✔ Protected Routes  
✔ Create Tasks  
✔ Delete Tasks  
✔ Logout System  
✔ REST API  

3. Installation

A) Backend
cd backend  
npm install  
npm start  

B) Frontend
cd frontend  
npm install  
npm run dev
